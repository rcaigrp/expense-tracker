# Expense Tracker

A simple module to track expenses.

## Goal
Track expenses and calculate totals.

## Acceptance Criteria
1. `expense_tracker` module exists and can be imported.
2. `add_expense` adds an expense and returns a dict.
3. `get_expenses` returns a list of expenses.
4. `calculate_total` returns the sum of all amounts.
