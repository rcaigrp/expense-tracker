# ExpenseTracker

A Python module to parse expense data and generate a summary report.

## Goal
Parse expense data (list of dicts) and generate a summary (total, by_category, by_date).

## Acceptance Criteria
1. `expense_tracker` module exists and can be imported.
2. `parse_expenses` function takes a list of expense dicts and returns a summary dict.
3. Tests pass and cover all criteria.
